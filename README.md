# SuperStore_Sales_Dashboard
## 1. Project Title / Headline
**SuperStore Analytics: Sales Performance & Forecasting Dashboard**
A dynamic, interactive Power BI dashboard built to analyze Superstore sales data, track KPIs, and forecast future revenue trends.
## 2. Short Description / Purpose
The SuperStore Sales Dashboard is a visually appealing and intuitive data visualization tool built to explore sales performance across different regions, categories, and segments. The dashboard focuses on time series analysis, sales forecasting, and KPI tracking. This tool is intended for business analysts and decision-makers who seek valuable insights and data-driven strategies to contribute to business success.
## 3. Tech Stack
The dashboard was built using the following tools and technologies:
- **Power BI Desktop** - Main data visualization platform for reporting and analysis
- **Power Query** - Data transformation and cleaning layer for reshaping raw Superstore data
- **DAX (Data Analysis Expressions)** - Used for calculated measures, KPIs, and time intelligence functions
- **Data Modeling** - Relationships established among tables for efficient analysis
- **Time Series Analysis** - For accurate sales forecasting
## 4. Objective
To contribute to the success of a business by utilizing data analysis techniques, specifically focusing on time series analysis to provide valuable insights and accurate sales forecasting.
## 5. Key Features
- **Sales Overview**: Track Total Sales, Profit, Orders, and Profit Margin
- **Time Series Forecasting**: Predict future sales trends using historical data
- **Regional Analysis**: Compare performance across different regions and states
- **Product Analysis**: Identify top-selling categories and sub-categories
- **Interactive Filters**: Slicers for Year, Region, Segment, and Category
- **Dark Theme UI**: Clean, modern interface for better readability
